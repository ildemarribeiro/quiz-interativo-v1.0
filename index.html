Nota que, no entanto, é comum manter o elemento em inglês, pois é um termo técnico específico do HTML e não há uma tradução direta para português.

O elemento head representa uma coleção de metadados para o documento. Metadados são, como o próprio nome diz, dados sobre os dados.
